# math 数学函数
math函数库提供各种计算功能的函数
## sum
用于计算数组或者数字序列的总和
#### 参数API
| 序号   	|  参数类型  	|                 说明                 	|
|--------	|:----------:	|:------------------------------------:	|
|    1   	| 数组或序列 	| 可传入数组或纯数序列(如Number序列) 	|
| 返回值 	|    字符    	|             所有数的总和             	|

#### 示例1 计算一个数组的总和

```groovy
return math.sum([1,3,5,7,9]);
```
